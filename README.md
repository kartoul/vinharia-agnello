# Vinharia Agnello

### Sistema de gerenciamento e análise de vinhos.
---

O usuário poderá adicionar as informações do vinho para cadastro, dentre elas:
- Nome do vinho
- Tipo 
- Safra 
- Quantidade em estoque
---

### Como realizar o cadastro de vinhos?
Para realizar o cadastro, o usuário deverá inserir as informações em uma caixa de de input que aparecerá na tela.

Após o cadastro, aparecerá uma caixa dizendo "Cadastro realizado! Veja os detalhes no console."

---

### Como ver as informações do vinho no console?

Para ver as informações do vinho no console, basta clicar com o botão direito do mouse em qualquer lugar da tela, selecionar 'Inspecionar' e acessar a aba 'Console'. 

Ou apenas clicar na tecla 'F12'.
